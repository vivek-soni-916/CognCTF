Name of this project is CogzCTF.
CTF stands for Capture The Flag. It is online game mainly intended for Security Researchers or you can say Hackers.

Participant have to find the string (known as flag) from challenges (challenges are so interesting).
There are so many categories of challenges you can expect from our side: 
	-Web
	-OSINT
	-Cryptography
	-Reverse Engineering
	-Binary Exploitation
	-Misc

Every three hours new challenges will be added. (Note that CTF event will be of 2 days)
Top three winners will get certificate and prizes.

Participants are not supposed to do any kind of Cyber Attack on site.
Event will be started in Cognizance (Technical Fest of CHARUAST University).
